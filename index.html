How to film the first 3 seconds

The finished result must appear immediately. Visual proof comes before specifications.

1. Show the PayoffOpen on the finished case on wrist. Let the transformation read before you explain the specs.
2. Create the ContrastPlace the regular watch beside the empty metal case so the transformation is instantly understood.
3. Deliver the SnapShow the watch going into the case with a tight, satisfying installation shot.
4. Crown Still ScrollsClose up on the crown and show it scrolling smoothly, so viewers know the case does not block daily operation.
